# SOW-007

## Title

Create the first `git` Developer Dashboard skill.

## Objective

Bootstrap `skills/git` as a self-contained skill repo and ship the first nested command, `git.smart.folder`, for disposable smart-folder rebuild workflows over stacked ticket branches.

## Deliverables

- skill-local version, changelog, README, docs, and ticket records
- explicit MIT license and README license documentation
- nested command packaging for `dashboard git.smart.folder`
- Perl implementation module and example script
- disposable Git repository tests
- Docker-based verification with 100% statement and subroutine coverage
- release commit and push
